(async, static) request(req) → {object}

Make a http request. Request config: { method {string} the http method to invoke (default "GET") uri {string} the uri to request username {string} the username to authenticate the request (optional) password {string} the password to authentication the request (optional) body {string|object|Uint8Array} the request body (optional) headers {object} a map of headers to add to the request (optional) requestApi {string} one of "fetch" or "xhr" (default "fetch") resolveWithFullResponse {boolean} return full response if true, else body only (default false) rejectUnauthorized {boolean} whether or not to reject self-signed certificates (default true) } Response object: { body {string|object|Uint8Array} the response body if given statusCode {string} the response status code statusText {string} the response status text headers {object} a map of the response headers }
