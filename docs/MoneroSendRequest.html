Class: MoneroSendRequest

MoneroSendRequest(param1, param2, param3, param4)

Configures a request to send/sweep funds or create a payment URI.

Constructor

new MoneroSendRequest(param1, param2, param3, param4)

Generic request to transfer funds from a wallet.

Examples:

let request1 = new MoneroSendRequest({
   accountIndex: 0,
   address: "59aZULsUF3YN...",
   amount: new BigInteger("500000"),
   priority: MoneroSendPriority.NORMAL
});

let request2 = new MoneroSendRequest(0, "59aZULsUF3YN...", new BigInteger("500000"), MoneroSendPriority.NORMAL); request2.setDoNotRelay(true); // do not relay transaction to the network
Parameters:
Name Type Description
param1 object | number | string request configuration, source account index, or destination address
Properties
Name Type Description
accountIndex int source account index to transfer funds from
address string single destination address (required unless destinations config given separately)
amount BigInteger single destination amount (required unless destination config given separately or sweep request)
priority int transaction priority (optional, default MoneroSendPriority.NORMAL)
subaddressIndices Array.<int> source subaddress indices to transfer funds from (optional)
destinations Array.<MoneroDestination> transfer destinations with addresses and amounts (required unless address/amount given separately)
paymentId string transaction payment ID (optional)
fee BigInteger transaction fee (optional, note: currently ignored)
doNotRelay boolean do not relay the transaction to the network if true
unlockTime int number of confirmations before the recipient can spend the funds
canSplit boolean allow funds to be transferred using multiple transactions (optional)
note string note for the transaction saved in the wallet (optional)
recipientName string recipient name for the transaction saved in the wallet (optional)
belowAmount BigInteger for sweep requests, include outputs below this amount when sweeping wallet, account, subaddress, or all unlocked funds (optional)
sweepEachSubaddress boolean for sweep requests, sweep each subaddress individually instead of together if true (optional)
keyImage string key image to sweep (ignored except in sweepOutput() requests)
param2 string | BigInteger destination address or send amount
param3 BigInteger | number send amount or transaction priority
param4 number transaction priority
